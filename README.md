## 명령어


| 명령어 | 설명 |
| ------ | ----------- |
| Hello   | 인사를 합니다. |
| Cody | 온도와 날씨, 그리고 거기에 맞는 옷을 추천합니다. |
| Menu    | 메뉴를 추천해줍니다.|
| play title | 보이스 채널에서 title을 재생합니다 |

## 그 외


> 이 프로젝트는 API discord.py 를 기반으로 개발되었습니다. 개발에 사용한 패키지 목록은 다음과 같습니다.

    - discord
    - discord.ext
    - youtube_dl
    - requests
    - bs4 (BeautifulSoup)
    - urllib
    - ffmpeg
